## ðŸ§  Getting Started

To follow along with the course, you can clone the repository and switch to the specific day you are studying.

1. **Clone the repository:**
```bash
git clone https://github.com/your-username/repository-name.git
```


2. **Navigate to the folder:**
```bash
cd repository-name
```


3. **Switch to a specific day:**
```bash
git checkout day-1
```


4. **View the project:**
Open `index.html` in your preferred browser or use the **Live Server** extension in VS Code.
